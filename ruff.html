Module Description

This module provides integration with Ruff, a fast Python linter written in Rust. It offers functionality to perform code quality checks on Python codebases.

Available Functions

check

Runs the ruff check command for basic linting of Python code.

check-with-config

Runs the ruff check command with a custom configuration file for more tailored linting.
