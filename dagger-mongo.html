Module Purpose

This module provides MongoDB for use as a Dagger Service. It offers a simple interface to work with MongoDB containers.

Available Functions

  • base() - Returns a mongo container

Quality Improvement Suggestions

1. Add Version Configuration

Currently, there's no way to specify which MongoDB version to use. Add version configuration:

func (m *Mongo) WithVersion(version string) *Mongo {
    return &Mongo{
        base: m.base.WithEnvVariable("MONGODB_VERSION", version)
    }
}

2. Add Health Check Function

Implement a health check to ensure MongoDB is ready:

func (m *Mongo) WaitForReady() *Mongo {
    return &Mongo{
        base: m.base.WithExec([]string{
            "mongosh", 
            "--eval", 
            "db.adminCommand('ping')"
        })
    }
}

3. Add Common Configuration Options

Implement common MongoDB configuration options:

func (m *Mongo) WithAuthEnabled(username string, password string) *Mongo {
    return &Mongo{
        base: m.base.
            WithEnvVariable("MONGO_INITDB_ROOT_USERNAME", username).
            WithEnvVariable("MONGO_INITDB_ROOT_PASSWORD", password)
    }
}
