Module Functionality

This is a workspace module that provides basic workspace management functions:

  • check: Run the tests in the workspace
  • diff: Show the changes made to the workspace in unified diff format
  • read: Read the contents of a file in the workspace at the given path
  • reset: Reset the workspace to the original state
  • tree: List the files in the workspace in tree format
  • work: No description provided
  • write: Write the contents of a file in the workspace at the given path

func validatePath(path string) error {
    if path == "" {
        return fmt.Errorf("path cannot be empty")
    }
    if strings.Contains(path, "..") {
        return fmt.Errorf("path cannot contain parent directory references")
    }
    return nil
}
