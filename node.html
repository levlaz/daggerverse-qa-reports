Module Description

This module provides Node.js integration for Dagger pipelines, offering various functions for building, testing, and managing Node.js applications.

Available Functions

  • build - Build returns the Container with the source built.
  • container - Container returns Node container.
  • ctr - No description provided
  • lint - Lint returns the output of the lint command.
  • run - Run returns the container with the command set.
  • start - Start returns the output of the code executed.
  • test - Test returns the result of the test executed.
  • with-container - WithContainer returns Node container with the given container.
  • with-npm - WithNpm returns Node container with npm configured as package manager.
  • with-source - WithSource returns Node container with source and cache set in it.
  • with-version - WithVersion returns Node container with given image version.
  • with-yarn - WithYarn returns Node container with yarn configured as package manager.

Note: Functions 'publish' and 'install' were skipped due to unsupported types.
