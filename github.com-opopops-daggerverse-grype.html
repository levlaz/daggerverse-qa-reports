Module Description

The Grype module is a Dagger module that provides integration with the Grype vulnerability scanner. It allows users to scan containers, directories, and files for security vulnerabilities.

Available Functions:

  • container - Returns configured grype container
  • scan - General scan functionality
  • scan-directory - Scan a directory for vulnerabilities
  • scan-file - Scan a specific file
  • scan-image - Scan a container image
  • with-registry-auth - Authenticate with registry
  • with-scan - Scan (for chaining)
  • with-scan-directory - Scan dir (for chaining)
  • with-scan-file - Scan file (for chaining)
  • with-scan-image - Scan container image (for chaining)
