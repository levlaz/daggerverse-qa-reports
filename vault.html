Module Description

The Vault module provides integration with HashiCorp Vault, offering functionality to interact with Vault secrets and authentication mechanisms.

Functions

  • host: Base host configuration
  • jwt: JWT authentication handler
  • kvget: GetSecretJSON returns a Vault secret as a JSON string
  • namespace: Namespace configuration
  • read: Read returns a vault secret as a JSON string
  • write: Write writes a vault secret and returns the response as a JSON string
  • with-host: WithHost sets the host for the Vault client
  • with-jwtauth: JWT authentication configuration
  • with-namespace: WithNamespace sets the namespace for the Vault client
  • with-userpass-auth: WithUserpassAuth sets the userpass authentication for the Vault client
  • test-*: Various test functions for read/write operations

func TestVaultIntegration(t *testing.T) {
    ctx := context.Background()
    
    // Start test vault server
    vaultServer := testutils.NewTestVault(t)
    defer vaultServer.Stop()

    // Test full workflow
    client := vault.New().
        WithHost(vaultServer.Address()).
        WithUserpassAuth("test", "test")

    // Write secret
    _, err := client.Write(ctx, "secret/test", map[string]string{
        "key": "value",
    })
    assert.NoError(t, err)

    // Read and verify
    secret, err := client.Read(ctx, "secret/test")
    assert.NoError(t, err)
    assert.Equal(t, "value", secret.Data["key"])
}
