Module Purpose

This module provides GraphQL schema manipulation capabilities. It allows you to:

  • Load GraphQL schemas from JSON specifications
  • Convert schemas to .graphql files

Available Functions

  • fromJson(spec: File) - Loads a GraphQL schema from a JSON specification file
  • file() - Converts the schema to a .graphql file
