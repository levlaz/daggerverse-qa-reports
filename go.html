Module Purpose

The Go module provides a comprehensive set of tools for building, testing, and managing Go projects within Dagger pipelines. It includes functionality for:

  • Building Go applications
  • Running tests
  • Managing dependencies
  • Configuring build environments
  • Performing linting

Available Functions

  • version() - Get Go version
  • source() - Access project source directory
  • moduleCache() - Manage Go module cache
  • buildCache() - Manage Go build cache
  • base() - Access base container
  • build() - Build Go packages
  • binary() - Build single main package
  • test() - Run tests
  • lint() - Lint the project
  • listPackages() - List Go packages
  • download() - Download dependencies
  • env() - Prepare build environment
