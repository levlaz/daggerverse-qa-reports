Module Description

A Go project tooling module that provides functionality for:

  • Setting up Go development environments
  • Managing Go versions
  • Linting Go code
  • Building Go projects

Available Functions:

  • version() - Get Go version
  • lintVersion() - Get Go linter version
  • source() - Get project source directory
  • base() - Build base container with Go
  • env() - Prepare build environment
  • lint() - Lint the project
