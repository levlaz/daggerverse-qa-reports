Module Purpose

The inline-node module provides functionality for running Node.js code and managing Node.js packages inline within Dagger pipelines.

Available Functions:

  • code(code: String!) - Executes provided Node.js code
  • ctr() - Returns the base container
  • withPackage(name: String!) - Adds a single NPM package
  • withPackages(packages: [String!]!) - Adds multiple NPM packages
