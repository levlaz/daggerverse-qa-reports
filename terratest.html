Module Purpose

This module is designed to run Terratest tests in a containerized environment. It provides functionality to execute Terraform tests using the Terratest framework with configurable Go and Terraform versions.

Available Functions

  • version() - Get the Golang image version
  • tfVersion() - Get the Terraform version
  • image() - Get the container image being used
  • withSource() - Set the source directory for tests
  • withCgoDisabled() - Disable CGO
  • withEnvVar() - Set environment variables
  • base() - Set up the container with Golang image and cache volumes
  • run() - Execute the Terratest tests
