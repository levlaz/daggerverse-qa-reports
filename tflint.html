Module Purpose

TFLint is a module designed to run Terraform linting checks. It provides functionality to initialize and run linting on Terraform code.

Available Functions

  • src() - Get the source directory
  • ctr() - Get the container configuration
  • base() - Set the base image and version
  • withSource() - Set the source directory
  • version() - Get TFLint version
  • run() - Execute TFLint commands
  • withInit() - Add init command configuration
  • runInit() - Execute init command
  • lint() - Configure lint operation
  • runLint() - Execute lint operation
