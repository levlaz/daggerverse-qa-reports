Module Description

This module provides test functionality for archive operations in Dagger. It includes tests for both archive files and directories with support for various compression formats.

Available Functions

  • Tests.all(): Executes all tests
  • Tests.archiveFiles(): Testing file archiving operations
  • Tests.archiveDirectory(): Testing directory archiving operations
  • ArchiveFiles: Methods for testing various archive formats (tar, zip, etc.)
  • ArchiveDirectory: Methods for testing directory archiving in various formats
