Module Functionality

This module provides a wrapper around the GitHub CLI (gh) with the following functions:

  • clone - Clone a GitHub repository
  • exec - Run a GitHub CLI command with arguments
  • pull-request - Work with GitHub pull requests
  • release - Manage releases
  • repo - Work with GitHub repositories
  • run - Run a GitHub CLI command as a string
  • source - Git repository source management
  • terminal - Open an interactive terminal
  • with-git-exec - Run git commands
  • with-repo - Set GitHub repository context
  • with-source - Load Git repository source
  • with-token - Set GitHub token

2. Implement Retry Logic

Add retry mechanism for network operations:


func withRetry(operation func() error, maxAttempts int) error {
    var err error
    for i := 0; i < maxAttempts; i++ {
        if err = operation(); err == nil {
            return nil
        }
        time.Sleep(time.Second * time.Duration(i+1))
    }
    return err
}
