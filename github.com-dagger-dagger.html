Module Description

The github.com/dagger/dagger module is the core development module for Dagger itself. It provides a comprehensive set of tools for developing, testing, and managing various components of the Dagger ecosystem including the CLI, engine, and SDKs.

Main Functions:

  • bench - Find and run benchmark suites
  • check - CI entrypoint for checking everything works
  • cli - Develop the Dagger CLI
  • dev - Create dev containers with CLI and engine
  • engine - Develop the Dagger engine container
  • generate - Run code generation for SDKs and docs
  • go - Go toolchain management
  • scripts - Run Dagger scripts
  • sdk - Develop Dagger SDKs
  • test - Find and run test suites
