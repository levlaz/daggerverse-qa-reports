Module Purpose & Functions

The Jumppad module is designed to test blueprints using either docker or podman. It provides functionality for managing and testing Jumppad configurations.

Available Functions:

  • withVersion() - Installs a specific version of jumppad from GitHub releases
  • withFile() - Installs a specific version of jumppad from the provided file
  • withCache() - Uses a specific cache volume for docker or podman server
  • testBlueprint() - Tests a blueprint using either docker or podman
  • testBlueprintWithVersion() - Tests a blueprint with a specific version of jumppad
  • testBlueprintWithBinary() - Tests a blueprint with an existing binary
