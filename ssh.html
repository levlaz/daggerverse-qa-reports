Module Description

This module provides SSH functionality for Dagger pipelines. It allows users to execute SSH commands and manage SSH connections.

Available Functions

  • baseCtr() - Returns the base container
  • command(args []string) - Executes SSH commands
  • destination() - Returns the SSH destination
  • generateKeys() - Generates SSH keys
  • opts() - Returns SSH options
