Module Purpose

This module provides SSH functionality for executing commands on remote servers using either password or identity file authentication.

Functions

Core Functions:

  • baseContainer() - Creates the base container for SSH operations
  • config() - Configures SSH connection settings
  • withPassword() - Sets up password authentication
  • withIdentityFile() - Sets up key-based authentication
  • command() - Executes commands on the remote server
  • container() - Returns a prepared container for SSH operations

Suggested Improvements

1. Add Connection Timeout Configuration

Currently missing connection timeout settings which could be crucial for production environments.

func (c *Config) WithTimeout(seconds int) *Config { return &Config{ timeout: seconds, ... } }

2. Implement Connection Retry Logic

Add retry mechanism for more robust connections in unstable networks.

func (c *Config) WithRetry(attempts int, delay time.Duration) *Config { return &Config{ retryAttempts: attempts, retryDelay: delay, ... } }

3. Add Proxy Support

Support for SSH proxy jumping would enhance the module's versatility.

func (c *Config) WithProxy(proxyHost string, proxyPort int) *Config { return &Config{ proxySettings: &ProxyConfig{ host: proxyHost, port: proxyPort, }, ... } }
