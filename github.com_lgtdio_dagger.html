Module Purpose

This module provides a development environment for the Dagger Engine. It includes tools for:

  • Building and testing the Dagger CLI
  • Managing the Dagger Engine
  • Developing multiple SDK implementations (Go, Python, TypeScript, etc.)
  • Documentation management
  • Running tests and benchmarks

Key Functions

  • DaggerDev - Main entry point for development tools
  • DaggerDevCli - CLI development and publishing tools
  • DaggerDevDaggerEngine - Engine container management
  • DaggerDevDocs - Documentation generation and deployment
  • DaggerDevTest - Test execution and management
  • DaggerDevSdk - SDK development tools for various languages
