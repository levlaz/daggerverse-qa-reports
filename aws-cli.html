Module Purpose

This module provides a generic interface to use AWS CLI in Dagger pipelines, along with high-level functions for AWS interactions. It's particularly useful for AWS operations within Dagger pipelines.

Available Functions:

  • container - Base container management
  • ecr - Amazon ECR operations
  • exec - Run AWS CLI commands
  • region - Region management
  • sts - AWS STS operations
  • with-config - Config file mounting
  • with-credentials - Credentials management
  • with-profile - Profile selection
  • with-region - Region setting
  • with-static-credentials - Static credentials management
  • with-temporary-credentials - Temporary credentials handling
  • without-static-credentials - Credentials removal

2. Add Error Retry Mechanism

Implement retries for AWS API calls to handle temporary failures.

func withRetry(operation func() error, maxAttempts int) error {
    var err error
    for attempt := 1; attempt <= maxAttempts; attempt++ {
        err = operation()
        if err == nil {
            return nil
        }
        time.Sleep(time.Second * time.Duration(attempt))
    }
    return err
}
