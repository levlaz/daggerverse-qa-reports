Module Purpose

This module provides functionality for building and serving Docusaurus documentation sites. It offers both development and production environments for Docusaurus projects.

Available Functions

  • base(): Returns base container for running docusaurus with docs mounted
  • build(): Builds production docs
  • serve(): Serves production docs locally as a service
  • serve-dev(): Builds and serves development docs as a service
  • cache-volume-name(): Helper function for cache management
  • dir(): Directory management function
  • disable-cache(): Cache control function
  • src(): Source management function
