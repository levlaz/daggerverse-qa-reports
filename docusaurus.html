Module Description

This module provides functionality for working with Docusaurus documentation sites. It includes capabilities for building, serving, and developing Docusaurus documentation.

Available Functions

  • cache-volume-name() - Cache volume management
  • dir() - Directory management
  • disable-cache() - Cache control functionality
  • serve() - Serves production docs locally as a service
