Module Functionality

This module provides functionality for testing Helm charts with various operations including installation, linting, packaging, and publishing.

Available Functions:

  • chart-package() - Packages the chart
  • chart-publish() - Publishes the chart
  • create() - Creates a new instance
  • lint() - Performs linting
  • login() - Handles authentication
  • package() - Packages the module
  • package-install() - Installs a package
  • push() - Pushes the package
