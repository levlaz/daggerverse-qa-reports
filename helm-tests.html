Module Purpose

This module provides a test suite for Helm chart operations, including various functionalities such as linting, packaging, publishing, and installation of Helm charts.

Functions

  • all() - All executes all tests
  • create() - Basic creation functionality
  • lint() - Performs linting of Helm charts
  • login() - Handles authentication
  • package() - Packages Helm charts
  • push() - Pushes Helm charts to repository
  • chartLint() - Specific chart linting functionality
  • chartPackage() - Specific chart packaging functionality
  • chartPublish() - Publishes charts to repository
  • chartInstall() - Installs charts to cluster
  • packageInstall() - Installs packages
  • installNamespace() - Handles namespace installation
