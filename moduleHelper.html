Module Purpose

This module serves as a reference implementation for basic module structure in Dagger. It provides helper functions for working with module schemas and SDK file paths.

Functions

  • get-module-schema: Get the schema of a module represented as an XML string
  • get-sdk-main-file: Get the main SDK file path for the given module name and SDK
  • get-type-name: (Note: Has unsupported types)
