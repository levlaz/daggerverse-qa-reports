Module Purpose

This module serves as a reference for basic module structure and provides utilities for working with Dagger modules. It includes functions for:

  • Getting module schemas in XML format
  • Retrieving SDK main file paths
  • Working with type names

Available Functions

  • getModuleSchema() - Get the schema of a module represented as an XML string
  • getSdkMainFile() - Get the main SDK file path for the given module name and SDK
  • getTypeName() - Get the name of a type definition
