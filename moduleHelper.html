Module Purpose

This module serves as a reference for basic module structure and provides utility functions for working with Dagger modules. It includes functions for:

  • Getting module schema as XML
  • Getting SDK main file paths
  • Getting type names from TypeDef objects

Function List

  • getModuleSchema(mod: Module!) -> String! - Get the schema of a module represented as an XML string
  • getSdkMainFile(sdk: String!, name: String!) -> String! - Get the main SDK file path for the given module name and SDK
  • getTypeName(type: TypeDef!) -> String! - Get type name from TypeDef
