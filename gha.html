Module Description

The GHA module provides functionality for working with GitHub Actions workflows. It allows users to generate, manage, and execute GitHub Actions workflows using Dagger.

Available Functions

  • generate - Generate GitHub Actions workflow
  • job - Manage GitHub Actions job
  • workflow - Add a workflow
  • workflows - Manage multiple workflows

Note: One function (with-workflow) was skipped due to unsupported types.
