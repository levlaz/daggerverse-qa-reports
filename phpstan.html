Module Purpose & Functions

PHPStan is a PHP Static Analysis Tool that helps discover bugs in your code without running it. The module provides the following functions:

  • analyse() - Analyzes source code
  • container() - Returns the base container
  • generateBaseline() - Generates a baseline file
