Module Purpose

The Concourse module provides functionality for managing and interacting with Concourse CI pipelines. It offers tools for:

  • Running Concourse databases
  • Managing pipelines and their configurations
  • Handling resources and resource types
  • Managing variables and secrets
  • Running quickstart clusters

Available Functions:

- container         
- database          Runs the Concourse database
- interpolate       
- load-pipeline     Load a pipeline configuration from a YAML configuration
- postgres          
- quickstart        Runs an all-in-one Concourse cluster
- resource          Initialize a resource. Resources represent external versioned assets
- resource-types    
- runtime           
- secret-vars       
- state-tag         
- vars              
- web-port          
- with-secret-var   Adds a secret to use for a Concourse config ((variable))
- with-var          Adds a variable to use for a Concourse config
