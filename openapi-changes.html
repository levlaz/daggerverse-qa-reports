Module Description

This module helps discover what changed between two OpenAPI specs, or a single spec over time. It supports OpenAPI 3.1, 3.0 and Swagger.

Available Functions

  • diff() - Compare two OpenAPI specs
  • local() - Check for changes in a local Git repository
  • remote() - Check for changes in a remote Git repository
