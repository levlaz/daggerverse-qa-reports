Module Purpose

A swiss army knife of functions for working with Rust projects. The module provides essential tools for Rust development including:

  • clippy() - Lints Rust code and detects common mistakes
  • format() - Formats Rust code using Rustfmt
  • formatCheck() - Verifies code formatting
  • audit() - Performs security audits using cargo-audit
