Module Purpose

This module is designed to integrate with Alpine Linux. It provides functionality for:

  • Creating Alpine Linux containers
  • Managing APK packages
  • Configuring architecture and branch settings
  • Customizing Alpine Linux distributions

Available Functions

  • distro() - Returns the Alpine distribution being used
  • arch() - Returns the hardware architecture being built for
  • branch() - Returns the Alpine branch used for packages
  • packages() - Returns the list of APK packages to install
  • container() - Builds an Alpine Linux container

func (m *Alpine) WithArch(ctx context.Context, arch string) (*Alpine, error) {
    validArchs := []string{"amd64", "arm64", "arm/v7"}
    for _, validArch := range validArchs {
        if arch == validArch {
            return m.arch(arch), nil
        }
    }
