Module Description

The Alpine module is designed to integrate with Alpine Linux. It provides functionality to create and customize Alpine Linux containers with specific configurations.

Available Functions:

  • arch() - The hardware architecture to build for
  • branch() - The Alpine branch to download packages from
  • container() - Build an Alpine Linux container
  • distro() - The distro to use
  • packages() - The APK packages to install
