Module Description

This module provides Java development environment capabilities with the following functions:

  • ctr() - Creates a base container for Java operations
  • withMaven(version) - Configures Maven version
  • run(args) - Executes Java commands
  • maven(args) - Runs Maven commands
  • withJdk(version) - Sets specific JDK version
  • withProject(source) - Incorporates a Java project source
