Module Description

This module provides Java development capabilities including JDK and Maven integration. It allows users to:

  • Set up Java environments with specific JDK versions
  • Integrate Maven for build management
  • Execute Java applications
  • Work with Java projects

Module Functions

  • ctr() - Creates a container for Java operations
  • withMaven(version: String) - Configures Maven with specified version
  • run(args: [String]) - Executes Java commands
  • maven(args: [String]) - Runs Maven commands
  • withJdk(version: String) - Sets specific JDK version
  • container() - Returns the configured container
  • withProject(source: Directory) - Sets up a Java project from source
