Module Functionality

This module provides Ngrok integration for Dagger, allowing you to expose local services through Ngrok tunnels. Available functions:

  • qr() - Return first Ngrok public url QR code
  • share() - Share a Service via Ngrok
  • test() - Test sharing via Ngrok with Nginx
  • url() - Retrieve first Ngrok public url
