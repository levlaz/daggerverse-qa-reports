Module Description

This module is designed to manage Kind (Kubernetes in Docker) clusters. It provides functionality for creating, managing, and interacting with local Kubernetes clusters using Kind.

Module Functions

  • container() - Container with the kind and docker binaries
  • version() - Prints the kind CLI version
  • getClusters() - Lists existing kind clusters by their name
  • createCluster() - Creates a local Kubernetes cluster
  • deleteCluster() - Deletes a cluster
  • kubeconfig() - Exports cluster kubeconfig
  • logs() - Exports logs to a directory

1. Add Input Validation

The module would benefit from adding input validation for cluster names and configuration parameters.

func validateClusterName(name string) error {
    if len(name) == 0 {
        return fmt.Errorf("cluster name cannot be empty")
    }
    if len(name) > 63 {
        return fmt.Errorf("cluster name must be less than 63 characters")
    }
    if !regexp.MustCompile(`^[a-z0-9][a-z0-9-]*[a-z0-9]$`).MatchString(name) {
        return fmt.Errorf("cluster name must consist of lower case alphanumeric characters or '-'")
    }
    return nil
}
