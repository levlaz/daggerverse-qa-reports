Module Purpose

This is a reference module that demonstrates basic module structure for getting started with Dagger. It serves as an example implementation with basic functionality.

Available Functions

containerEcho()

Returns a container that echoes whatever string argument is provided

Arguments:
  • stringArg (String!, required): The string to echo

grepDir()

Returns lines that match a pattern in the files of the provided Directory

Arguments:
  • directoryArg (Directory!, required): A directory to search in
  • pattern (String!, required): The pattern to search for
