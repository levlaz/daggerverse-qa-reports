Module Purpose

This module demonstrates how any package manager that supports pyproject.toml with a build backend (PEP 517) naturally works with Dagger. Python modules are installed with `pip install .` by the runtime container.

Available Functions

  • containerEcho(stringArg: String!): Echo a string in an alpine container
  • grepDir(directoryArg: Directory!, pattern: String!): Find a pattern in a directory
