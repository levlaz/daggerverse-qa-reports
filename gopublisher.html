Module Purpose

The gopublisher module is designed to help publish Go modules. It provides various functions for managing Go module publishing workflows.

Available Functions

  • src() - Get the source directory
  • ctr() - Get the base container
  • base() - Set the base container
  • terminal() - Get a terminal for the container
  • goModPath() - Get the module path
  • goModVersion() - Get the module version
  • goModPublish() - Publish the module
  • withEnvVariable() - Set environment variables
  • withCgodisabled() - Disable CGO
  • withGit() - Install Git
  • withCurl() - Install cURL
  • withSource() - Set the source directory
  • withEnvVarsFromStrs() - Set multiple environment variables
