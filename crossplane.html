Module Description

This module provides functionality for working with Crossplane packages. It includes tools for initializing, packaging, and pushing Crossplane packages, as well as handling custom templates and configurations.

Available Functions

  • get-xplane-container: Returns the default image for helm
  • init-custom-package: Initializes Crossplane Package based on custom templates and a configuration file
  • init-package: Initializes Crossplane Package
  • package: Packages Crossplane Package
  • push: Pushes Crossplane Package
