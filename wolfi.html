Module Description

Wolfi is a container-native Linux distribution with an emphasis on security. The module provides functionality to work with Wolfi Linux containers.

Module Functions

  • container() - Build a Wolfi Linux container
    • Arguments:
      • packages: [String!] - APK packages to install
      • arch: String - Hardware architecture to target
      • overlays: [Container!] - Overlay images to merge on top of the base
