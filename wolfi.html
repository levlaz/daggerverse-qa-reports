Module Description

Wolfi is a container-native Linux distribution with an emphasis on security.

Module Functions

  • container() - Build a Wolfi Linux container
    • Parameters:
      • packages [String!] - APK packages to install
      • arch String - Hardware architecture to target
      • overlays [Container!] - Overlay images to merge on top of the base
