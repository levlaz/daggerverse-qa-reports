Module Purpose & Functions

This module is designed for PHP SDK development, providing tools for testing, linting, and formatting PHP code.

Available Functions:

  • test() - Run tests in source directory (accepts optional group parameter)
  • lint() - Run linter in source directory
  • format() - Return diff from formatting source directory
  • formatStdout() - Return stdout from formatting source directory

2. Add Input Validation

The module could benefit from stronger input validation:

// Add to the test function definition:
test: {
    group?: string
    if group != _|_ {
        // Validate that group name follows PHPUnit group naming convention
        check: {
            valid: strings.MaxRunes(group, 100) && 
                   strings.MinRunes(group, 1) &&
                   match(group, "^[a-zA-Z0-9_-]+$")
        }
    }
}
