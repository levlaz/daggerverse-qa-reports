Module Purpose

This module is designed for PHP SDK development, providing tools for testing, linting, and formatting PHP code. It includes the following functions:

  • test() - Run tests in source directory
  • lint() - Run linter in source directory
  • format() - Return diff from formatting source directory
  • formatStdout() - Return stdout from formatting source directory
  • base() - Build a PHP container with SDK source files and dependencies

Quality Improvement Suggestions

1. Add Input Validation

Add validation for the source directory input to ensure it contains valid PHP files.

func validateSource(source dagger.Directory) error {
    // Check if composer.json exists
    _, err := source.File("composer.json")
    if err != nil {
        return fmt.Errorf("composer.json not found in source directory")
    }
    
    // Verify PHP files exist
    files, err := source.Entries()
    if err != nil {
        return err
    }
    
    hasPhpFiles := false
    for _, file := range files {
        if strings.HasSuffix(file, ".php") {
            hasPhpFiles = true
            break
        }
    }
    
    if !hasPhpFiles {
        return fmt.Errorf("no PHP files found in source directory")
    }
    
    return nil
}
