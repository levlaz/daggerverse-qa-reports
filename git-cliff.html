Module Description

git-cliff is a module for generating changelogs from git history. It provides various functions for managing changelog generation, version bumping, and git repository interaction.

Module Functions

  • bumped-version: Prints bumped version for unreleased changes
  • run: Run git-cliff with all options previously provided
  • with-bump: Bump the version for unreleased changes
  • with-config: Sets the configuration file
  • with-current: Processes the commits that belong to the current tag
  • with-env-variable: Adds an environment variable to the git-cliff container
  • with-gitea-token: Sets the Gitea API token
  • with-github-token: Sets the GitHub API token
  • with-gitlab-token: Sets the GitLab API token
  • with-latest: Processes the commits starting from the latest tag
  • with-netrc: Add netrc credentials
  • with-output: Writes output to the given file
  • with-prepend: Prepends entries to the given changelog file
  • with-repository: Sets the git repository
  • with-secret-variable: Adds an env variable containing a secret
  • with-skip-commit: Sets commits that will be skipped in the changelog
  • with-strip: Strips the given parts from the changelog
  • with-topo-order: Sorts the tags topologically
  • with-unreleased: Processes the commits that do not belong to a tag
