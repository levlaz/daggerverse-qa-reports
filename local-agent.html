Module Purpose & Functions

This module provides functionality for creating and managing local development environments.

Available Functions:

  • devEnvironment(source: Directory!) - Creates a development environment for the project, installs all the needed tools and libraries
  • workOn(source: Directory!) - Enter a development environment and export the workspace directory
