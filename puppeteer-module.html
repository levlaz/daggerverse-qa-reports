Module Purpose and Functions

This module provides a set of tools for web automation and screenshot capture using Puppeteer. It allows users to interact with web pages, capture screenshots, and perform various web automation tasks.

Available Functions:

  • captureScreenshot() - Capture a screenshot of a webpage
  • clickAndCapture() - Click an element and capture a screenshot
  • scrollAndCapture() - Scroll and capture multiple screenshots
  • fillForm() - Interact with a form - fill in fields and submit
  • captureTitle() - Get the page title of a webpage
  • puppeteerContainer() - Create a container with Puppeteer and its dependencies
