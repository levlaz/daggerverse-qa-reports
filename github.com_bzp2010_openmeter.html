Module Description

The OpenMeter module provides CI/CD pipelines and build tools for the OpenMeter project. It includes functionality for building binaries, running tests, generating SDKs, and performing various lint checks.

Functions:

  • ✅ build - Build individual artifacts for testing and development
  • ✅ ci - CI pipeline execution
  • ✅ etoe - End-to-end testing
  • ✅ generate - Generate various artifacts (Python, Node, Web SDKs)
  • ✅ lint - Perform various lint checks
  • ✅ quickstart-test - Run quickstart tests
  • ✅ release - Handle release processes
  • ✅ snapshot - Create snapshots
  • ✅ test - Run test suites
