Module Description

This module provides functionality to run a k3s server that can be accessed both locally and in pipelines. It offers tools for managing Kubernetes clusters and executing various Kubernetes-related operations.

Functions

  • config() - Returns the config file for the k3s cluster
  • container() - Base container operations
  • kns() - Runs k9s on the target k3s cluster
  • kubectl() - Runs kubectl on the target k3s cluster
  • server() - Returns a newly initialized kind cluster
  • withContainer() - Returns a newly initialized kind cluster with custom container configuration

2. Implement Retry Mechanism

Add retry logic for cluster operations to handle transient failures.

func withRetry(maxAttempts int, op func() error) error {
    var err error
    for i := 0; i < maxAttempts; i++ {
        if err = op(); err == nil {
            return nil
        }
        time.Sleep(time.Second * time.Duration(i+1))
    }
    return fmt.Errorf("operation failed after %d attempts: %v", maxAttempts, err)
}
