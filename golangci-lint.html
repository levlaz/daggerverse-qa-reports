Module Purpose

This module is designed to integrate golangci-lint, a fast Go linters runner, into Dagger pipelines. It provides functionality to run lint checks on Go code.

Functions

containerWithSource(src Directory)

Returns a container with the source code ready for linting.

lint(dir Directory)

Runs golangci-lint on the specified directory and returns the results.
