Module Purpose

The Linguist module is primarily used for confirming that a .gitattributes file correctly marks a given file as being generated. It provides functionality to analyze source code using GitHub's Linguist tool.

Module Functions

  • src() - The directory containing the source code to analyze
  • repo() - The Linguist GitHub repository to build
  • commit() - The Git commit for the Linguist repository to use
  • run() - Run the github-linguist tool on the source code with options for:
    • file analysis
    • git revision analysis
    • language breakdown
    • JSON output
