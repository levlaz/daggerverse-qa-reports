Module Purpose

This module provides functionality for working with Protocol Buffers (Protobuf) files using the Buf tool. It offers the following functions:

  • push() - Push Protobuf files to the Buf Registry
  • format() - Format Protobuf files
  • lint() - Lint Protobuf files

Suggestions for Improvement

1. Add Input Validation

Add validation for the source directory to ensure it contains valid Protobuf files.

func validateProtoFiles(src *Directory) error {
    files, err := src.Entries(ctx)
    if err != nil {
        return err
    }
    
    hasProto := false
    for _, file := range files {
        if strings.HasSuffix(file, ".proto") {
            hasProto = true
            break
        }
    }
    
    if !hasProto {
        return fmt.Errorf("no .proto files found in source directory")
    }
    return nil
}
