Module Purpose

Module wrapping the gscloud commandline tool. The module provides functionality to interact with GridScale's cloud infrastructure, particularly focused on Kubernetes cluster configuration.

Functions

  • kubeconfig()
    • Returns the generated kubeconfig file
    • Required parameters: userId, userToken (Secret), clusterUuid
  • container()
    • Returns container with gscloud binary located at /usr/bin/gscloud

1. Add Input Validation

Add validation for the GsCloud parameters to ensure proper formatting and prevent errors.


func (g *Gscloud) validateInputs(userId string, clusterUuid string) error {
    if len(userId) == 0 {
        return fmt.Errorf("userId cannot be empty")
    }
    if !regexp.MustCompile(`^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$`).MatchString(clusterUuid) {
        return fmt.Errorf("invalid cluster UUID format")
    }
    return nil
}

2. Add Error Handling and Retry Logic

Implement retry logic for API calls to handle temporary failures.


func (g *Gscloud) withRetry(maxAttempts int, fn func() error) error {
    var err error
    for i := 0; i < maxAttempts; i++ {
        if err = fn(); err == nil {
            return nil
        }
        time.Sleep(time.Second * time.Duration(i+1))
    }
    return fmt.Errorf("failed after %d attempts: %v", maxAttempts, err)
}
