Module Description

This module provides test functionality for Kafka integration, focusing on connection testing in both single-node and cluster environments.

Available Functions:

  • all() - Executes all tests
  • singleNodeConnect() - Tests connection to a single Kafka node
  • clusterConnect() - Tests connection to a Kafka cluster
