Module Purpose

This module is designed to automatically generate markdown documentation for Helm charts. It provides functionality to generate documentation from requirements and values files.

Available Functions

  • generate() - Generates markdown documentation for Helm charts from requirements and values files
    • Parameters:
      • chart: Directory! - A directory containing a Helm chart
      • templates: [File!] - A list of Go template files for rendering documentation
      • sortValuesOrder: String - Order to sort values table ("alphanum" or "file")
