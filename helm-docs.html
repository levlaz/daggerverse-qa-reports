Module Purpose

This module is designed to automatically generate markdown documentation for Helm charts from requirements and values files.

Available Functions

generate()

Generate markdown documentation for Helm charts from requirements and values files.

Arguments:
  • chart (Directory!) - A directory containing a Helm chart
  • templates ([File!]) - A list of Go template files to use for rendering the documentation
  • sortValuesOrder (String) - Order in which to sort the values table ("alphanum" or "file")
