Improvement Suggestions

1. Add Version Configuration

The module should allow users to specify which Drupal version they want to use.


func (d *Drupal) WithVersion(version string) *Drupal {
    // Validate version
    if version == "" {
        version = "10.1.0"  // default version
    }
    d.version = version
    return d
}

2. Add Composer Integration

Include methods to manage Composer dependencies


func (d *Drupal) WithComposerPackage(package string) *Drupal {
    return d.WithExec([]string{"composer", "require", package})
}

3. Add Development Tools

Include common development tools for Drupal development


func (d *Drupal) WithDevTools() *Drupal {
    return d.WithExec([]string{
        "apt-get update && apt-get install -y",
        "xdebug",
        "drush",
        "phpunit"
    })
}
