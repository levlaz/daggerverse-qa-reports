Module Description

This module provides functionality to interact with surge.sh, a static web hosting platform. It allows users to publish directories to surge.sh directly from their Dagger pipelines.

Available Functions

  • base() - Returns base image with surge installed
  • get-token() - Get token interactively
  • publish() - Publish directory to surge.sh

2. Add Proper Error Handling and Retries

Implement retry mechanism for publish operations to handle temporary network issues.

func (s *Surge) publishWithRetry(ctx context.Context, directory *Directory, domain string) (*Container, error) {
    maxRetries := 3
    for i := 0; i < maxRetries; i++ {
        result, err := s.publish(ctx, directory, domain)
        if err == nil {
            return result, nil
        }
        if i < maxRetries-1 {
            time.Sleep(time.Second * time.Duration(i+1))
            continue
        }
        return nil, fmt.Errorf("failed to publish after %d retries: %w", maxRetries, err)
    }
    return nil, fmt.Errorf("unexpected error in retry loop")
}
