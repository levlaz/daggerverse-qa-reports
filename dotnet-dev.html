Module Purpose

This module contains functions for developing the .NET SDK, including running tests, generating introspection, and formatting code.

Available Functions

  • container() - Returns the base container
  • generate(introspectionJson) - Generate code from introspection JSON file
  • test(introspectionJson) - Testing the SDK
  • lint() - Lint all CSharp source files in the SDK
  • check(introspectionJson) - Run test and lint
  • format() - Format all CSharp source files
