Module Purpose

This module provides integration with Bash Automated Testing System (Bats). It allows users to run Bats tests within their Dagger pipelines.

Available Functions:

  • withSource() - Mounts a source directory for testing
  • run() - Executes Bats tests with specified arguments

1. Add Version Validation

The module should validate the Bats version parameter to ensure compatibility.


func validateVersion(version string) error {
    validVersions := []string{"1.0.0", "1.1.0", "1.2.0"}
    for _, v := range validVersions {
        if version == v {
            return nil
        }
    }
    return fmt.Errorf("unsupported Bats version: %s", version)
}
