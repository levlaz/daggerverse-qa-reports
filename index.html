Module Purpose

A collection of functions for building, formatting, testing, linting and scanning your Go project for vulnerabilities.

Functions:

  • bench() - Execute benchmarks defined within the target project
  • build() - Build a static binary from a Go project using the provided configuration
  • container() - Base container used by all golang dagger functions
  • format() - Format the source code using gofumpt
  • lint() - Lint the project using golangci-lint
  • modVersion() - Get the Go version from go.mod
  • test() - Execute tests defined within the target project
  • vulncheck() - Scan for vulnerabilities using govulncheck
  • withPrivate() - Enable private Go module support
  • withPrivateLoad() - Load existing .netrc configuration
