Module Functionality

This module provides a comprehensive suite of Go development tools and utilities. The following functions are available:

  • bench: Execute benchmarks defined within the target project, excludes all other tests
  • build: Build a static binary from a Go project using the provided configuration
  • format: Format the source code within a target project using gofumpt
  • lint: Lint the target project using golangci-lint
  • mod-version: Echoes the version of go defined within a projects go.mod file
  • test: Execute tests defined within the target project, ignores benchmarks by default
  • vulncheck: Scans the target project for vulnerabilities using govulncheck
  • with-private: Enable private Go module support by dynamically constructing a .netrc auto-login
  • with-private-load: Enable private Go module support by loading an existing .netrc auto-login configuration
