Module Functions

This module provides a variety of testing utilities for visualizing and testing Dagger execution:

  • accounting() - Sleeps for 1 second then 2 seconds, useful for UI testing
  • deepSleep() - Sleeps forever
  • spam() - Logs current time in a tight loop
  • streamingLogs()/streamingChunks() - Print batches of logs on intervals
  • fail()/failEffect()/failLog() - Various failure testing utilities
  • colors16()/colors256() - Color testing utilities
  • echo()/uppercase() - String manipulation utilities
  • terminal() - Terminal-related testing
  • service() - Service testing utilities
  • countFiles() - Directory file counting utility
