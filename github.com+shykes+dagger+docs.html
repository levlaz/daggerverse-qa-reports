Module Purpose

This module is designed for managing and generating documentation for the Dagger project. It provides functionality for:

  • Building documentation website
  • Generating API schema and CLI reference docs
  • Linting documentation files
  • Deploying documentation to Netlify
  • Managing PHP SDK API documentation
