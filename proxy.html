Module Purpose & Functions

This module provides proxy service functionality for Dagger pipelines. It allows you to create and manage proxy services within your Dagger workflows.

Available Functions:

  • ctr() - An OCI-compatible container interface
  • service() - Get the proxy Service
  • with-service() - Add a service to proxy
