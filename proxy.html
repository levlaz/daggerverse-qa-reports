Module Purpose

This module provides proxy functionality to forward multiple services into a single service with multiple ports.

Available Functions:

  • ctr() - Returns an OCI-compatible container
  • withService(service, name, frontend, backend) - Adds a service to proxy
  • service() - Gets the proxy Service
