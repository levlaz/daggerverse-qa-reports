Module Functionality

This module provides proxy functionality for Dagger pipelines. It allows you to set up and manage proxy services within your Dagger workflows.

Available Functions:

  • ctr - An OCI-compatible container, also known as a Docker container.
  • service - Get the proxy Service
  • with-service - Add a service to proxy
