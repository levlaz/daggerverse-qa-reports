Module Purpose

The PowerShell Script Analyzer (PSScriptAnalyzer) for Dagger. This module provides functionality to analyze PowerShell scripts for potential issues and best practices violations.

Available Functions:

  • check(file: File!, excludeRules: [String!]) - Analyzes a PowerShell script file and returns a report
  • base() - Returns the base container configuration
