Module Purpose

This module is designed to help in developing the Elixir SDK for Dagger. It provides various development tools and utilities for testing, linting, and generating SDK code.

Available Functions

  • test() - Test the SDK
  • lint() - Lint the SDK
  • generate() - Generate the SDK API
  • sdkTest() - Run the SDK tests
  • codegenTest() - Run dagger_codegen tests
  • syncImage() - Sync Elixir image for dev and runtime consistency
  • withBase() - Set up base configuration
  • withCodegen() - Configure codegen settings
