Module Purpose

This module is designed to help with developing the Elixir SDK for Dagger. It provides various utilities for testing, linting, and generating SDK code.

Available Functions:

  • test() - Test the SDK
  • lint() - Lint the SDK
  • generate() - Generate the SDK API
  • sdk-test() - Run the SDK tests
  • codegen-test() - Run dagger_codegen tests
  • with-base() - Set up base container with SDK source
  • with-codegen() - Configure container for code generation
