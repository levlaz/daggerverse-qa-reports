Module Purpose

This module creates a configuration file for OCI registry authentication and returns it as a Secret that can be safely mounted into a Container. It addresses the security concerns of storing registry credentials in the filesystem, which could leak into Dagger's layer cache.

Available Functions

  • secret() - Create the registry configuration.
  • secret-mount() - Create a SecretMount for mounting the registry configuration into a container.
  • with-registry-auth() - Add credentials for a registry.
  • without-registry-auth() - Removes credentials for a registry.
