Module Purpose

This module provides tools for securely handling OCI registry authentication. It creates a configuration file that can be safely mounted as a Secret into a Container, avoiding the security risks of storing credentials in the filesystem that could be exposed through Dagger's layer cache.

Module Functions

  • secret() - Create the registry configuration
  • secret-mount() - Create a SecretMount for registry configuration
  • with-registry-auth() - Add credentials for a registry
  • without-registry-auth() - Removes credentials for a registry

2. Implement Retry Mechanism

Add retry logic for registry operations to handle temporary network issues.

func withRetry(operation func() error, maxAttempts int) error {
    var lastErr error
    for attempt := 1; attempt <= maxAttempts; attempt++ {
        if err := operation(); err != nil {
            lastErr = err
            time.Sleep(time.Second * time.Duration(attempt))
            continue
        }
        return nil
    }
    return lastErr
}
