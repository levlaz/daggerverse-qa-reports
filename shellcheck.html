Module Purpose

The shellcheck module provides shell script analysis capabilities using the ShellCheck tool. It allows checking shell scripts for potential issues and provides various reporting formats.

Available Functions

Main Functions:

  • check(file: File) - Reports issues that need to be fixed in a given shell script file

Report Functions:

  • issues() - Returns array of detected issues
  • jSon() - Returns JSON format of the report
  • report() - Returns human-readable report
  • fixedDiff() - Shows differences in fixed version
  • assert() - Validates if there are any issues
  • fixed() - Returns fixed version of the file
