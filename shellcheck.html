Module Purpose

This module provides shell script analysis functionality using shellcheck. It offers the following capabilities:

  • Analyzing shell scripts for potential issues
  • Generating detailed reports of found issues
  • Providing fixes and diffs for identified problems
  • Offering both JSON and human-readable output formats

Available Functions

- check(): Report the issues that need to be fixed on a given file
- Report object methods:
  - issues(): Get list of found issues
  - jSON(): Get JSON format report
  - report(): Get human readable report
  - fixedDiff(): Get diff of fixes
  - assert(): Verify no issues exist
  - fixed(): Get fixed file content
