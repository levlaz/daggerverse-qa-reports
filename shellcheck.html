Module Functionality

This module provides shellcheck functionality with the following main features:

  • Check shell scripts for syntax and semantic issues
  • Generate reports in different formats (JSON, plain text)
  • Provide detailed issue information including line numbers and error codes
  • Offer suggestions for fixing detected issues

Available Functions

  • check(file: File) - Main function to perform shell script analysis
  • issues() - Get list of detected issues
  • jSon() - Get report in JSON format
  • report() - Get human-readable report
  • fixedDiff() - Get diff of applied fixes
  • assert() - Verify if the script passes checks
  • fixed() - Get fixed version of the script
