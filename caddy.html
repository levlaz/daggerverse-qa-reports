Module Description

This module provides Caddy server functionality in Dagger. It includes capabilities for container management, Caddy file configuration, and service operations.

Available Functions

  • container - Manages Caddy container operations
  • get-caddy-file - Retrieves Caddy configuration file
  • serve - Starts the Caddy server
  • services - Manages Caddy services
  • with-service - Configures Caddy with specific service

func (m *Module) Serve(ctx context.Context) error {
    container, err := m.Container(ctx)
    if err != nil {
