Module Purpose & Functions

This module provides functionality for managing Caddy server configurations and services. It allows users to set up and manage reverse proxy configurations using Caddy.

Available Functions:

  • services() - Returns a list of service configurations
  • withService() - Adds a new service configuration
  • getCaddyFile() - Retrieves the Caddy configuration file
  • container() - Returns the Caddy container
  • serve() - Starts the Caddy service
