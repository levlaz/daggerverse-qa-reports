Module Purpose

Git-Changelog is a command-line tool that allows you to generate a changelog from a git repository.

Available Functions

  • changelog - Generates a changelog for a git repository and version
  • chglog-dir - Returns the directory containing chglog config and template
  • container - Returns a container with git-chglog installed and configured
  • image-repository - Gets the image repository information
  • image-tag - Gets the image tag information
