Module Description

This module provides test functionality for archiving operations. It includes three main components:

  • Tests:
    • all() - Executes all tests
    • archiveDirectory() - Directory archiving tests
    • archiveFiles() - File archiving tests
  • ArchiveFiles: Provides various archive format testing for files including:
    • tar(), tarBr(), tarBz2(), tarGz(), tarLz4(), tarSz(), tarXz(), tarZst(), zip()
  • ArchiveDirectory: Provides various archive format testing for directories including:
    • tar(), tarBr(), tarBz2(), tarGz(), tarLz4(), tarSz(), tarXz(), tarZst(), zip()
