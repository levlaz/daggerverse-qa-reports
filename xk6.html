Module Purpose

The xk6 module is a Dagger module designed to build custom k6 binaries. k6 is an open-source load testing tool that makes performance testing easy and productive for engineering teams.

Available Functions

  • build(): Build a custom k6 binary
  • container(): Container operations
  • with-platform(): Set GOOS, GOARCH and GOARM environment variables for cross-compilation
