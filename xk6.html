Module Functionality

This module provides functionality for building custom k6 binaries. k6 is an open-source load testing tool that makes performance testing easy and productive for engineering teams.

Available Functions:

  • build() - Build a custom k6 binary
  • container() - Container management functionality
  • with-platform() - Set GOOS, GOARCH and GOARM environment variables for cross-compilation
