Module Purpose

This module provides several useful submodules for working with:

  • .NET projects (dotnet submodule)
  • Docker Compose operations (compose submodule)
  • Helm chart management (helm submodule)
  • Testing utilities (testing submodule)

Functions

Core Functions:

  • with-ca - Add an additional CA certificate
  • with-cred - Add additional credentials
  • with-downloaded-ca - Download and add CA certificate

Dotnet Submodule:

  • restore - Restore dependencies
  • build - Build .NET project
  • test - Run tests
  • publish - Publish with runtime
  • buildContainer - Build container with runtime

Helm Submodule:

  • lint - Run helm lint
  • schema - Run helm-schema validation
  • docs - Generate documentation
  • unittest - Run helm unit tests
  • package - Package Helm chart
  • template - Template Helm chart
  • push - Push to registry
