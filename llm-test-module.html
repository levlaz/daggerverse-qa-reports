Module Purpose

This module is designed for testing allow-llm policy enforcement around git-sourced modules and is called by its parent LLMDirModuleDepender. It provides functionality to interact with LLM APIs in a minimal way.

Module Functions

  • prompt(stringArg: String!): access the LLM api in the most minimal way: prompt it, then retrieve history
  • save(stringArg: String!): this is a hack until we can return the LLM type and replay history
