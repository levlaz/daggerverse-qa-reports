Module Purpose

This module provides a set of functions to work with Apache Avro Tools. It helps in converting Avro IDL files to Avro schema files and provides access to Apache Avro Tools functionality.

Available Functions:

  • avro-tools - Get a container with Apache Avro Tools installed
  • idl - Convert an Avro IDL file to Avro schema file
  • idl-2-schemata - Convert an Avro IDL file to Avro schema files in the provided directory
