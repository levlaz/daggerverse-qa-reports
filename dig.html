Module Purpose

The dig module provides DNS query functionality through a simple interface. It allows users to perform various DNS record lookups including A, AAAA, MX, NS, and TXT records, as well as DNS trace operations and propagation verification.

Available Functions

  • a(domain) - Performs A record lookup
  • aaaa(domain) - Performs AAAA record lookup
  • mx(domain) - Performs MX record lookup
  • ns(domain) - Performs NS record lookup
  • txt(domain) - Performs TXT record lookup
  • trace(domain) - Performs DNS resolution trace
  • verifyPropagation(domain, expectedValue) - Checks DNS record propagation
