Module Purpose

This module provides a testing framework to verify dagger container structure. It offers utilities for testing container outputs, checking file existence, and verifying directory structures.

Functions

  • assertOutput() - Test a container output
  • getExitCode() - Returns a container exit code
  • assertFileExistence() - Verify file existence with optional permission checks
  • isDir() - Check if a path is a directory
