Module Purpose

This module contains functions for developing the .NET SDK, including:

  • Running tests
  • Generating introspection
  • Code formatting
  • Linting

Available Functions

  • container() - Returns the base container
  • generate() - Generate code from introspection json file
  • test() - Testing the SDK
  • lint() - Lint all CSharp source files in the SDK
  • check() - Run test and lint
  • format() - Format all CSharp source files
